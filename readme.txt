Para configurar o banco de dados é necessário editar o arquivo /config/app.php
alterando os valores das linhas : 251, 258, 259, 260.

Sistema produzido com Cakephp framework e bootstrap Front-End framework
Sistema está responsivo e adaptado para todos os tamanhos de tela.

O sistema contém:
Tela inicial - Lista todos os comentário do banco de dados.
Tela de Login - Para autenticação do Usuário
Tela de Registro - Para criação de um novo Usuário
Área do usuário - Para editar informações do cadastro e inserir um foto.
Tela de inserção para novos comentário
Tela de Edição/Exclusão de um comentário
Formulário de busca de comentário pelo Autor.

O administrador pode alterar ou apagar qualquer um dos comentários,
para ser administrador dever-se alterar para 1 o valor que por padrão é 0
na tabela Usuários do banco de dados.

A senhas estão sendo criptografados assim que o cadastro é efetuado.

Para apresentação final do projeto a base de dados já está com4 usuários
sendo um administrador e os demais comum e alguns comentários.
